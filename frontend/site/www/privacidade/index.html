Política de Privacidade

Última atualização: Janeiro 2025

O aplicativo "Converse com Maria" respeita sua privacidade e está comprometido em proteger seus dados pessoais. Esta política descreve como coletamos, usamos e protegemos suas informações.

📋 Dados que Coletamos

  • • Nome: Como você deseja ser chamado nas conversas
  • • Gênero: Para personalizar as respostas
  • • Estado civil e informação sobre filhos: Para contextualizar o aconselhamento
  • • Mensagens enviadas no chat: Para gerar respostas personalizadas
  • • Dados de uso: Estatísticas anônimas para melhorar o aplicativo

🎯 Como Usamos seus Dados

  • • Personalizar a experiência de conversa com Maria
  • • Adaptar as respostas ao seu perfil e necessidades
  • • Melhorar continuamente o serviço oferecido
  • • Enviar notificações de lembretes de oração (se autorizado)

🔒 Segurança dos Dados

Seus dados são armazenados de forma segura:

  • • Dados locais são armazenados no seu dispositivo
  • • Comunicação criptografada com nossos servidores
  • • Não compartilhamos suas informações pessoais com terceiros
  • • Não vendemos seus dados para fins publicitários

🌐 Serviços de Terceiros

O aplicativo utiliza os seguintes serviços de terceiros:

  • • Groq (LLaMA): Inteligência Artificial para gerar as respostas de Maria
  • • Google Cloud Text-to-Speech: Para gerar a voz de Maria
  • • Google AdMob: Para exibição de anúncios (versão gratuita)
  • • Firebase: Para autenticação e armazenamento de dados
  • • Stripe: Para processamento de pagamentos (assinaturas Premium)
  • • MercadoPago: Para processamento de pagamentos no Brasil

Esses serviços possuem suas próprias políticas de privacidade. Recomendamos a leitura das mesmas para entender como seus dados são tratados por cada provedor.

🗑️ Exclusão de Dados

Você tem o direito de solicitar a exclusão de todos os seus dados a qualquer momento. Para isso:

  • 1. Acesse o menu de Configurações no aplicativo
  • 2. Selecione "Excluir meus dados"
  • 3. Ou envie um e-mail para contato@conversecommaria.com.br

👶 Privacidade de Menores

O aplicativo não é direcionado a menores de 13 anos. Não coletamos intencionalmente informações de crianças. Se você é pai ou responsável e acredita que seu filho nos forneceu informações pessoais, entre em contato conosco.

📝 Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Alterações significativas serão comunicadas através do aplicativo. Recomendamos revisar esta página regularmente.
